# Telecom Customer Churn Analysis Dashboard

This project analyzes telecom customer churn using Power BI.

## Tools Used
- Power BI
- DAX
- Power Query
- Excel / CSV Dataset

## Key Insights
- Churn rate analysis
- Customer behavior patterns
- Revenue impact due to churn
- Contract type and service analysis
